# Wazuh Security Monitoring Project

## Overview
This project demonstrates **security monitoring and threat detection** using **Wazuh**, an open-source SIEM and XDR platform. It focuses on monitoring endpoints, detecting suspicious activity, and analyzing security alerts.

## Project Goal
- Monitor system logs and security events  
- Detect threats and anomalies  
- Gain hands-on SIEM and IDS experience  

## Technologies Used
- Wazuh  
- Elastic Stack  
- Kibana  
- Kali Linux  

## Documentation
Detailed setup, configuration, and analysis steps are in **[Wazuh.md](Wazuh.md)**.

## Features
- Endpoint log monitoring  
- Intrusion detection  
- File integrity monitoring  
- Vulnerability detection
