# Wazuh SOC Live Monitoring Project

## Objective
This project demonstrates a hands-on setup of a **Wazuh Security Operations Center (SOC)** with:
- **Manager:** Kali Linux
- **Agent:** Parrot OS
- Live monitoring of security events

It covers installation, configuration, and real-time monitoring using Wazuh and Kibana dashboards.

## Tools Used
- **Wazuh Manager & Agent**
- **Kali Linux** (Manager)
- **Parrot OS** (Agent)
- **Kibana Dashboard** for visualizing alerts
- Linux Syslogs / Event Logs

## Project Structure
1. **Config-Files/** – Manager and agent configuration files  
2. **Installation-Scripts/** – Commands/scripts for installation  
3. **Logs-Alerts/** – Sample alerts captured during live monitoring  
4. **Screenshots/** – Dashboard and agent monitoring images  
5. **Documentation/** – Step-by-step setup guide

## Key Learning Outcomes
- Installed and configured Wazuh manager and agent on Linux systems  
- Connected agent to manager for centralized log collection  
- Executed live monitoring and captured real-time security events  
- Analyzed alerts and visualized data on Kibana dashboards  
- Documented configuration and monitoring processes
